How to use this panel
When a customer pays:
1. Enter their name + WhatsApp number above
2. Click "Generate code" — a unique code is created
3. Click "Send on WhatsApp" — message is pre-filled
4. Customer enters code at access.html
5. They get redirected to the app automatically
Managing codes:
— Revoke a code to instantly block access
— Reactivate a revoked code if needed
— Uses shows how many times code was entered
— All data stored in this browser only
— Export CSV to keep a backup
Important: Codes are stored in this browser's localStorage. Use the same browser and device to manage codes. Export a CSV backup regularly. If you clear your browser data, all codes will be lost.
