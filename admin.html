How to use this panel
When a customer pays:
1. Confirm payment in your STC Pay / bank app
2. Enter their name + WhatsApp number above
3. Click Generate — unique code is created
4. Click "Send on WhatsApp" — message auto-fills
5. Customer enters code at access.html → gets app
Managing codes:
— Revoke — instantly blocks that customer's access
— Activate — re-enables a revoked code
— WA button — resend code to customer
— Uses — how many times code was entered
— Export CSV regularly as a backup
Codes are stored in this browser only. Always use the same device and browser to manage codes. Export a CSV backup weekly. To change your admin password: edit this file on GitHub, find ADMIN_PASSWORD and change the value.
